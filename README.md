## User Guide

##### Search screen

enter query to search
enter `q` to quit

##### Media selection screen

enter number of video

##### Media selection screen

Hotkeys:
* `q`: quit to search screen
* `n`: next song
* `N`: previous song
* `[`: scroll to previous page of playlist
* `]`: scroll to next page of playlist
